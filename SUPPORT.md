# Support

## Where to get help

- **Questions and ideas:** Use [GitHub Discussions](https://github.com/KaillouxWebDev/homelab-control-center/discussions) for general questions, usage tips, and feature ideas. This keeps issues focused on bugs and concrete changes.
- **Bugs:** Open an [issue](https://github.com/KaillouxWebDev/homelab-control-center/issues) and follow the bug report template. Include OS, Node version, Next.js version (see `package.json`, e.g. 14.2.18), Portainer version, and steps to reproduce when relevant.
- **Feature requests:** Open an issue with a clear description of the problem and proposed solution. See [CONTRIBUTING.md](CONTRIBUTING.md) for more on contributing.

## Maintainer time

This project is maintained on a best-effort basis. We appreciate your patience; addressing issues and reviewing contributions may take time. Pull requests and clear, reproducible bug reports help a lot.
